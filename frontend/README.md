# ⚛️ Frontend - React App

This is the web interface. Don't worry if you don't know React yet!

## Quick Start

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Files (Simplified)

```
src/
├── App.jsx              ← Main page (like main.py)
├── components/
│   ├── AQIForm.jsx      ← The input form
│   └── AQIResult.jsx    ← The colored result
└── index.css            ← Styles (Tailwind CSS)
```

## For Python Developers

| React Concept | Python Equivalent |
|--------------|-------------------|
| Component | Function that returns HTML |
| State (`useState`) | Variable that updates the UI |
| `fetch()` | `requests.post()` |
| `props` | Function arguments |
